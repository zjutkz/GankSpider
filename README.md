#GankSpider

It is a python network spider based on [scrapy](https://github.com/scrapy/scrapy).

#Usage

#####Notice that it is only works on MAC

Just using following command:

```
scrapy crawl gank
```

You will see images directory and enjot it ðŸ˜Š

You can set the range of images in setting.py by changing the value of PULL_RANGE:

```

```
